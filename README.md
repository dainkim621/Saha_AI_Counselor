# 🏛️ 사하구청 AI 상담사 프로젝트
사하구청 홈페이지 데이터를 활용한 지능형 민원 상담 서비스입니다.

## 📅 프로젝트 일정
-화요일: 팀플 회의

## 🛠️ 개발 환경
- OS: WSL2 (Ubuntu 22.04, 24.04)
- Language: Python 3.10(recommanded)
- Tools: Git, GitHub, Docker, Vscode

## 🛠️ Installation
### 1. Repository Clone

```bash
git clone <repository_url>
cd <project_folder>
```

### 2. Python Environment Setup

#### Option 1: venv

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

#### Option 2: Conda

```bash
conda create -n project python=3.10
conda activate project
pip install -r requirements.txt
```

### 3. Notes

* Python 3.10 환경을 권장합니다.
* 모든 패키지는 `requirements.txt` 기준으로 설치합니다.
* WSL 환경에서 개발하는 것을 권장합니다.
